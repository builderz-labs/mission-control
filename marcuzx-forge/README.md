# Marcuzx Forge

Where Systems Are Forged by AI.

This folder hosts the first in-repo MVP of the Marcuzx Forge ecosystem. Public platform branding uses Marcuzx Forge. Internal agent and memory references use Eak AI Factory.

## Modules
- `control` - orchestration control plane and entrypoint
- `agents` - Eak AI Factory role specifications
- `standards` - templates and engineering conventions
- `registry` - machine-readable platform and project metadata
- `memory` - institutional learning and reusable patterns
- `observatory` - observability contract and UI references
- `projects` - project snapshots discovered from the current workspace
