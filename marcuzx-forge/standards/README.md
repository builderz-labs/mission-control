# Marcuzx Forge Standards

This module provides the reusable conventions and templates future Marcuzx Forge projects should copy or adapt.

## Reuse Rules
- Start from the templates in `templates/`.
- Keep the standard docs contract intact unless a repo has a justified exception.
- Update changelog, decisions, and tasks together when platform structure changes.
- Keep agent specs evidence-driven and stage-aware.

## Contents
- `conventions.md` - platform-wide engineering conventions
- `repo-standards.md` - baseline repo expectations
- `templates/repo-doc-template.md` - standard doc contract
- `templates/architecture-template.md` - architecture template
- `templates/runbook-template.md` - runbook template
- `templates/task-template.md` - task system template
- `templates/changelog-template.md` - changelog template
- `templates/agent-spec-template.md` - agent contract template
