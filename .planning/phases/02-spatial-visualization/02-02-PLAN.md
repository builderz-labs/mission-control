# Plan 02-02: Custom Agent Node Component + Edge Types

## Objective
Create the custom React Flow node and edge components that render agent cards with status badges and typed relationship edges.

## Context
- Requirements: SPAT-01 (agent nodes with name/status), SPAT-02 (relationship edges), SPAT-05 (click opens detail)

## Tasks

### Task 1: Custom AgentNode component
Create `src/components/spatial/agent-node.tsx`:

```typescript
'use client'
import { Handle, Position, type NodeProps } from '@xyflow/react'
import type { AgentNodeData } from '@/store/canvas-store'
import { cn } from '@/lib/utils'

const STATUS_COLORS = {
  online: 'bg-emerald-500',
  idle: 'bg-emerald-500',  // map idle → online for display
  offline: 'bg-gray-500',
  busy: 'bg-amber-500',
  error: 'bg-red-500',
} as const

export function AgentNodeComponent({ data, selected }: NodeProps) {
  const d = data as AgentNodeData
  return (
    <div className={cn(
      "px-4 py-3 rounded-xl border bg-card/90 backdrop-blur-sm shadow-lg min-w-[160px] transition-all",
      selected ? "ring-2 ring-primary border-primary" : "border-border/50",
    )}>
      <Handle type="target" position={Position.Top} className="!bg-primary/50 !w-2 !h-2" />
      <div className="flex items-center gap-2 mb-1">
        <div className={cn("w-2.5 h-2.5 rounded-full", STATUS_COLORS[d.status] || 'bg-gray-500')} />
        <span className="font-semibold text-sm truncate">{d.label}</span>
      </div>
      {d.role && (
        <div className="text-[10px] text-muted-foreground uppercase tracking-wider truncate">{d.role}</div>
      )}
      <Handle type="source" position={Position.Bottom} className="!bg-primary/50 !w-2 !h-2" />
    </div>
  )
}
```

### Task 3: Custom edge components

Three edge styles for delegation, communication, supervision:

Use React Flow's `BaseEdge` + `getStraightPath` or `getBezierPath`.
