# Plan 06-01: PAD Emotional Model + Cognitive Biases

## Objective
Extend persona-engine with PAD emotional model (pleasure/arousal/dominance), exponential decay, and 8 cognitive biases with trait-based activation.

## Context
- Requirements: PRSA-04, PRSA-05, PRSA-06
- Existing: persona-engine.ts has BigFive (0-1 scale), MentalState with string[] emotions, buildSystemPrompt()
- Existing: agent-memory.ts with observation/reflection/relationship types
- Agent config stored as JSON in agents.config column

## Tasks

### Task 1: PAD Emotional Model types + functions
Extend `src/lib/persona-engine.ts`:

**New types:**
```typescript
interface PADState {
  pleasure: number    // -1 to 1
  arousal: number     // -1 to 1
  dominance: number   // -1 to 1
  updated_at: number  // unix epoch
}
```

**New functions:**
- `getPADState(db, agentId): PADState` — extract from agent config, default neutral (0,0,0)
- `updatePADState(db, agentId, delta: Partial<PADState>)` — merge into config
- `decayPADToBaseline(pad: PADState, bigFive: BigFive, elapsedMs: number): PADState` — exponential decay toward Big Five baseline
  - Baseline mapping: pleasure ← agreeableness, arousal ← extraversion, dominance ← (1-neuroticism)
  - Decay formula: `current + (baseline - current) * (1 - Math.exp(-elapsedMs / HALF_LIFE))`
  - HALF_LIFE = 30 minutes (1800000ms)
- `padToEmotionLabel(pad: PADState): string` — map PAD vector to emotion label (happy, angry, fearful, calm, excited, etc.)

### Task 2: Cognitive Bias Catalog
Add to `src/lib/persona-engine.ts`:

**Types:**
```typescript
interface CognitiveBias {
  name: string
  description: string
  activationCheck: (bigFive: BigFive, context?: string) => boolean
  promptFragment: string  // injected into system prompt when active
}
```

**8 biases with trait-based activation:**
1. Confirmation bias — low openness (< 0.3)
2. Anchoring — high conscientiousness (> 0.7)
3. Availability heuristic — high neuroticism (> 0.6)
4. Sunk cost — low openness + high conscientiousness
5. Bandwagon — high agreeableness (> 0.7) + high extraversion (> 0.6)
6. Dunning-Kruger — low openness + high extraversion
7. Status quo — low openness (< 0.3) + high neuroticism (> 0.5)
8. Recency — high neuroticism (> 0.6) + low conscientiousness (< 0.4)

**New function:**
- `getActiveBiases(bigFive: BigFive): CognitiveBias[]` — returns all biases whose activation condition is met

### Task 3: Integrate PAD + biases into buildSystemPrompt
Update `buildSystemPrompt()` to:
- Call `decayPADToBaseline()` with time since last update
- Call `padToEmotionLabel()` for human-readable emotion
- Call `getActiveBiases()` and append bias prompt fragments
- Include PAD values and active biases in the system prompt section

### Task 4: Unit tests
Create/extend `src/lib/__tests__/persona-engine.test.ts`:
- PAD state CRUD (get/update/defaults)
- Decay toward baseline (verify convergence)
- Emotion label mapping (8 octants)
- Bias activation for each of the 8 biases
- buildSystemPrompt includes PAD and bias sections
