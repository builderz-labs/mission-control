# Plan 06-02: Pairwise Trust + Drift Prevention

## Objective
Inter-agent pairwise trust scoring and persona re-injection drift prevention.

## Context
- Requirements: PRSA-07, PRSA-09
- Existing: agent_trust_scores table (security-based, per-agent, NOT pairwise)
- Existing: agent_relationships table (delegation/communication/supervision)
- Existing: updateMentalState() merges cognitive state into config JSON

## Tasks

### Task 1: Pairwise trust table migration
Add to `src/lib/phase-migrations.ts`:

```sql
CREATE TABLE IF NOT EXISTS agent_pairwise_trust (
  source_agent_id INTEGER NOT NULL REFERENCES agents(id) ON DELETE CASCADE,
  target_agent_id INTEGER NOT NULL REFERENCES agents(id) ON DELETE CASCADE,
  trust_score REAL NOT NULL DEFAULT 0.5,
  interaction_count INTEGER NOT NULL DEFAULT 0,
  last_interaction_at INTEGER,
  workspace_id INTEGER NOT NULL DEFAULT 1,
  updated_at INTEGER NOT NULL DEFAULT (unixepoch()),
  PRIMARY KEY (source_agent_id, target_agent_id)
);
CREATE INDEX IF NOT EXISTS idx_pairwise_trust_source ON agent_pairwise_trust(source_agent_id);
CREATE INDEX IF NOT EXISTS idx_pairwise_trust_target ON agent_pairwise_trust(target_agent_id);
```

### Task 2: Trust scoring functions
Create trust functions in `src/lib/persona-engine.ts`:

- `getPairwiseTrust(db, sourceId, targetId): { trust_score: number; interaction_count: number }` — get or default 0.5
- `updatePairwiseTrust(db, sourceId, targetId, delta: number): number` — add delta (clamped 0-1), increment interaction_count, broadcast event
- `getAgentTrustNetwork(db, agentId): Array<{ agent_id, agent_name, trust_score, interaction_count }>` — all trust relationships for an agent

### Task 3: Persona drift prevention
Add to `src/lib/persona-engine.ts`:

- `shouldReinjectPersona(turnCount: number, interval: number = 5): boolean` — returns true every N turns
- `measureDrift(original: BigFive, current: BigFive): number` — Euclidean distance in 5D space
- `buildReinjectablePrompt(db, agentId, turnCount): string` — wraps buildSystemPrompt with re-injection header when shouldReinject is true

### Task 4: Unit tests
Extend `src/lib/__tests__/persona-engine.test.ts`:
- Pairwise trust CRUD (get/update/default/clamp)
- Trust network retrieval
- Drift measurement (identical=0, opposite=max)
- Re-injection trigger (every 5 turns)
- Re-injection prompt includes persona reinforcement
