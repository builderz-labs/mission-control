# Plan 04-01: Teams Schema + @mention Routing Engine

## Objective
Teams table, team membership, and @mention routing logic that connects parsed mentions to EventBus events.

## Context
- Requirements: CHAT-01, CHAT-02, CHAT-03, CHAT-05
- Existing: `parseMentions()` and `resolveMentionRecipients()` in mentions.ts
- Existing: `chat.mention.routed` and `chat.mention.response` events already in EventDataMap but unused
- Pattern: Phase migrations in `src/lib/phase-migrations.ts`

## Tasks

### Task 1: Migrations for teams + team_members
Add to `src/lib/phase-migrations.ts`:
- `phase_048_teams`: `teams` table (id, name, description, workspace_id, created_at, updated_at)
- `phase_049_team_members`: `team_members` table (team_id FK, agent_id FK, joined_at) with UNIQUE constraint

### Task 2: Extend mentions.ts with team + special targets
Modify `src/lib/mentions.ts`:
- Add `@all` resolution → returns all active agents in workspace
- Add `@team:name` resolution → looks up team by name, returns member agents
- Add `@human` resolution → returns special `{ type: 'human' }` recipient
- Update `getMentionTargets()` to include teams in autocomplete results

### Task 3: Create mention-router.ts
Create `src/lib/mention-router.ts`:
- `routeMentions(db, messageId, content, conversationId, workspaceId)`: parses + resolves + broadcasts
- For each agent recipient: broadcast `chat.mention.routed` event
- For `@human`: create notification via existing `createNotification()`
- Returns `{ routed: string[]; unresolved: string[] }`
- Per-thread turn counting: query message count in conversation where from_agent is agent, reject if > 3 agent-to-agent turns without human

### Task 4: Add loop prevention EventBus events
Add to EventDataMap:
- `chat.mention.loop_prevented`: `{ conversationId: string; agentName: string; turnCount: number }`

### Task 5: Unit tests
Create `src/lib/__tests__/mention-router.test.ts`:
- Test @all routing
- Test @team:name routing
- Test @human notification
- Test loop prevention (4th agent turn blocked)
- Test unresolved mentions
