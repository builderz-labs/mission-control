# Plan 05-03: Debate Panel UI

## Objective
Debate room panel with argument tree, voting interface, and real-time SSE updates.

## Context
- Requirements: DEBT-06, DEBT-08
- Existing: workflow-panel.tsx pattern (tabs, status badges, detail views)
- EventBus events: debate.created, debate.round.started, debate.round.completed, debate.argument.submitted, debate.vote.cast, debate.concluded

## Tasks

### Task 1: Create DebatePanel component
Create `src/components/panels/debate-panel.tsx`:

**DebatesTab (list view):**
- Fetch debates from GET /api/debates
- Filter by status (all/propose/critique/rebut/vote/concluded)
- Each row: topic, participant count, current round/phase, status badge, "View" button
- "New Debate" button → form (topic, select participants, max rounds, token budget)

**DebateDetailView:**
- Header: topic, status badge, round X of Y, token budget meter (used/total)
- Argument tree: grouped by round, then by phase (propose/critique/rebut)
  - Each argument: agent avatar, agent name, content, confidence badge, timestamp
- Voting section (visible during vote phase):
  - Vote tally bar (accept vs reject)
  - Each vote: agent name, vote badge, reason
- Action buttons:
  - "Advance Phase" (when all arguments submitted for current phase)
  - Outcome display when concluded

### Task 2: Register panel route
Add `case 'debates':` to page.tsx ContentRouter.
