    Add after the `getDatabase()` function:

    ```typescript
    /**
     * Execute a write operation inside a BEGIN IMMEDIATE transaction.
     * Acquires the write lock immediately at transaction start, reducing
     * contention when multiple concurrent requests attempt writes.
     *
     * Usage:
     *   writeTransaction(db, (tx) => {
     *     tx.prepare('INSERT INTO ...').run(...)
     *     tx.prepare('UPDATE ...').run(...)
     *   })
     */
    export function writeTransaction<T>(
      database: Database.Database,
      fn: (db: Database.Database) => T
    ): T {
      const begin = database.prepare('BEGIN IMMEDIATE')
      const commit = database.prepare('COMMIT')
      const rollback = database.prepare('ROLLBACK')

      begin.run()
      try {
        const result = fn(database)
        commit.run()
        return result
      } catch (err) {
        rollback.run()
        throw err
      }
    }
    ```

    CRITICAL: Do NOT change any existing code in db.ts. Only ADD the new function. Existing callers will be migrated to use writeTransaction() in later phases as they are touched.

    CRITICAL: The prepared statements for BEGIN/COMMIT/ROLLBACK are created per-call intentionally — better-sqlite3 prepared statements are lightweight and this avoids stale statement issues.
