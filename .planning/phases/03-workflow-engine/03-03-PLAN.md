# Plan 03-03: Workflow Panel UI

## Objective
Dashboard panel showing workflow templates, active runs with phase progress, and approval controls.

## Context
- Requirements: WKFL-06, WKFL-07, WKFL-09
- Existing: Panel pattern from agent-squad-panel.tsx, task-board-panel.tsx
- API: /api/workflows (templates), /api/workflows/runs (runs), advance/approve/reject routes

## Tasks

### Task 1: WorkflowPanel component
Create `src/components/panels/workflow-panel.tsx`:
- Two tabs: "Templates" and "Active Runs"
- Templates tab: list templates with phase count, create button
- Runs tab: list active runs with status badge, current phase indicator

### Task 2: WorkflowRunDetail sub-component
Show run detail with:
- Phase progress bar (completed/current/pending phases)
- Current phase output preview
- Approve/Reject buttons when phase is paused
- Phase history with timestamps

### Task 3: CreateWorkflowDialog
Modal for creating workflow template:
- Name, description fields
- Phase builder: add/remove/reorder phases with drag
- Per-phase: name, agent_role, requires_approval toggle, input/output schema (textarea for JSON)

### Task 4: Register panel
Add `case 'workflows':` to ContentRouter returning `<WorkflowPanel />`.
