# Cross Repo Learning

Marcuzx Forge captures reusable patterns from each discovered project through the registry, project summaries, decisions, and memory patterns. Each future repo should contribute:

- architecture snapshots
- decisions that changed delivery behavior
- reusable implementation patterns
- recurring failure modes
- PR summaries tied back to the originating project

The first learning sources in this workspace are `mission-control`, `ai-orchestrator`, and `crewAI`.
